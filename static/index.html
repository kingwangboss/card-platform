卡密验证接口

验证卡密

POST /api/cards/verify

验证卡密是否有效，如果未激活则自动激活。此接口不需要认证。

请求体:
{
  "card_number": "1pZKyLAywgUOEUqd"  // 要验证的卡密号码
}
响应示例 (未激活卡密):
{
  "message": "Card activated successfully",
  "card": {
    "card_number": "1pZKyLAywgUOEUqd",
    "duration_days": 30,
    "is_activated": true,
    "activated_at_str": "2024-01-01T12:00:00Z",
    "expires_at_str": "2024-01-31T12:00:00Z"
  }
}
响应示例 (已激活且有效的卡密):
{
  "card_number": "1pZKyLAywgUOEUqd",
  "duration_days": 30,
  "is_activated": true,
  "activated_at_str": "2024-01-01T12:00:00Z",
  "expires_at_str": "2024-01-31T12:00:00Z"
}
错误响应示例:
{
  "error": "Card expired",
  "message": "This card has expired",
  "card": {
    "card_number": "1pZKyLAywgUOEUqd",
    "duration_days": 30,
    "is_activated": true,
    "activated_at_str": "2024-01-01T12:00:00Z",
    "expires_at_str": "2024-01-31T12:00:00Z"
  }
}
可能的错误响应:
  • 404 Not Found: 卡密不存在
  • 400 Bad Request: 卡密已过期
  • 500 Internal Server Error: 服务器内部错误
注意事项:
  • 卡密一旦激活就开始计时，无法暂停或重置
  • 过期的卡密无法重新激活
  • 卡密有效期从激活时间开始计算
